# Modern SaaS Dashboard

1. Install dependencies:
   npm install

2. Run development server:
   npm run dev

Yêu cầu: Node 18+, pnpm/npm/yarn.

Thông tin:
- Next.js 14 (App Router)
- TypeScript strict
- Tailwind CSS
- Framer Motion
- Three.js + @react-three/fiber + drei
- Recharts
- next-themes, lucide-react, react-hot-toast, date-fns
